# EtharaAI HRMS Lite Backend

Node.js + TypeScript + Express + MongoDB (Mongoose)

## Features
- Employee CRUD
- Attendance CRUD
- RESTful API
- Server-side validation
- Error handling
- DTOs
- Modular structure (controllers, services, repositories, models)
- Unit tests

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start development server:
   ```bash
   npm run dev
   ```
